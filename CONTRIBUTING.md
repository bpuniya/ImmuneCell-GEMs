# Contributing

Thanks for your interest in contributing!

## What you can contribute
- Add new immune cell models with provenance
- Add new media definitions
- Add new validation tests and notebooks
- Improve docs, tutorials, and figures

## Ground rules
- Every model must have an entry in `metadata/models.csv`
- Provide citation/provenance for each model
- Prefer JSON models for portability
- Keep tests lightweight and reproducible
